Las listas no ordenadas se pueden hacer con asteriscos, signos menos o signos más:


## Su sintáxis es:

### Con asteriscos:

```markdown
* Item 1
    * Item 1.1
* Item 2
* Item 3
```
### Con signos menos:

```markdown
- Item 1
    - Item 1.1
- Item 2
- Item 3
```

### Con signos más:

```markdown
+ Item 1
    + Item 1.1
+ Item 2
+ Item 3
```


## ¿Cómo se ven?

### Con asteriscos:

* Item 1
  * Item 1.1
* Item 2
* Item 3
  
### Con signos menos:

- Item 1
  - Item 1.1
- Item 2
- Item 3
  
### Con signos más:

+ Item 1
  + Item 1.1
+ Item 2
+ Item 3

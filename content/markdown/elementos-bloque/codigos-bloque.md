Los códigos de bloque son fragmentos de código o texto preformateado que se muestran en un bloque de texto separado del resto del texto.

## Su sintáxis es

```markdown
def hola_mundo():
    print('Hola mundo!')
```

## ¿Cómo se ve?

```python
def hola_mundo():
    print('Hola mundo!')
```

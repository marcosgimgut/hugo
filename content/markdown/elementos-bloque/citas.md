Las citas son bloques de texto que se utilizan para resaltar una parte del texto.
En markdonw, las citas se pueden crear con el caracter `>`:

## Su sintáxis es:

```markdown
> Esto es una cita.
```

## ¿Cómo se ve?

> Esto es una cita.

Los elementos de énfasis son aquellos que sirven para resaltar una parte del texto. Los más comunes son los siguientes: negrita, cursiva y tachado.

## Su sintáxis es:

Negrita:

```markdown
**Esto es negrita**
```

Cursiva:

```markdown
*Esto es cursiva*
```

Tachado:

```markdown
~~Esto está tachado~~
```

## ¿Cómo se ve?

**Esto es negrita**

*Esto es cursiva*

~~Esto está tachado~~
